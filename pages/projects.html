01 / Project In Production Testing

SafeVision AI

Industrial safety compliance platform built to replace a paid vendor at AMNS.

A computer-vision system monitoring 10+ live cameras for PPE compliance — helmets, vests, restricted-zone breaches, and crowd density — with automated alerts and a full compliance dashboard. Trained 4+ YOLO models reaching up to 97% mAP50. Built to give AMNS more control and configurability than their existing third-party vendor offered, at no recurring license cost.

Problem & Solution

Problem

Industrial sites rely on manual supervision for PPE compliance and unsafe-behavior detection. This is slow, inconsistent, and impossible to scale across dozens of cameras and shifts. AMNS was paying recurring licensing costs to a third-party vendor (DARSA) for a system with limited functionality and no configurability.

Solution

An in-house computer vision safety platform across 10+ AI cameras, currently in final testing to replace the third-party vendor entirely. 4+ YOLO models trained, achieving up to 97% mAP50 and 95%+ average precision/recall across helmet, safety vest, and restricted-zone violation classes.

Challenges

Balancing detection latency vs accuracy across model sizes on limited GPU compute (Kaggle free-tier 2×T4)
Enforcing strict class balance and excluding the ambiguous "head" class to reduce false positives
Using overlap-ratio-based matching instead of IoU for small PPE bounding boxes

Learnings

Production CV is 20% model, 80% data pipeline and matching logic
Temporal consistency streaks reduce false positive alerts significantly
Operator UX matters as much as model accuracy — both need to work together

Solution

04 / Project

FinAI

Mutual fund portfolio rebalancing, scored and explained.

Platform computing a real-time Rebalance Score (0–100) from a weighted 5-factor model — portfolio drift, volatility, risk alignment, tax efficiency, and market momentum — fed into a Bayesian decision engine. Includes a fund recommendation engine with STCG/LTCG tax-zone detection and an interactive what-if allocation simulator.

5factor model
4decision zones
0–100score range
PythonFastAPIReactSupabaseBayesian ML
Problem & Solution

Problem

Retail investors struggle to keep portfolios aligned with their risk profile as markets drift. Off-the-shelf tools are too simplistic or locked behind enterprise pricing.

Solution

A platform computing a real-time Rebalance Score using a weighted 5-factor model and Bayesian decision engine that classifies portfolios into Hold/Watch/Rebalance/Urgent zones with explainable rationale.

Challenges

Modeling realistic transaction costs: spreads, taxes, lot tracking
Producing suggestions users can trust and reason about — not a black-box score

Learnings

In finance, explainability beats marginal accuracy — users need to understand the "why"
Clean data contracts between scoring and recommendation engines make systems far easier to extend
